**Built with CMake and Modern C++ (C++17/20)**

This project implements efficient algorithms for processing interval queries on arrays, including range maximum queries, prefix sum calculations, and optimized counting sort.

## Project Structure

project/
├── CMakeLists.txt # CMake build configuration
├── headers/ # Header files
│ ├── array_generator.h # Test array generation
│ ├── array_io.h # Array input/output operations
│ ├── array_max.h # Maximum element algorithms
│ ├── array_prefix_sum.h # Prefix sum computations
│ ├── counting_sort.h # Counting sort implementation
│ └── interval_queries.h # Interval query processing
├── src/ # Source files
│ ├── array_generator.cpp
│ ├── array_io.cpp
│ ├── array_max.cpp
│ ├── array_prefix_sum.cpp
│ ├── counting_sort.cpp
│ ├── interval_queries.cpp
│ └── main.cpp # Program entry point
└── README.md # Documentation
